How to setup Spacemacs for clojure in windows
  • 1. Install Emacs

    Open www.emacs.org and click on the windows download button. Navigate to the main gnu ftp website. (link to main emacs ftp site ). Click on the folder with the latest emacs version number on it (emacs-26 as of this writing). If you are on windows XP then download the emacs-XX-i686.zip . If you are on windows 7 or windows 10 download emacs-XX-x86_64.zip. Unzip the file in any of the folder of your choice.
